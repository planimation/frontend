Description

The Planimation API provides programmatic access to Planimation, an online PDDL problem solver and visualiser. Planimation with user interface can accessed through here. Planimation API supports the same functionality that the user interface supports.

Planimation currently works in 2 ways. The user can sumbit either 3 PDDL files, which express the domain, problem and animation profile respectively, or 1 VFG file and a receive a certain format of visualisation (so far, GIF, PNG, WEBM and MP4 are supported). VFG is a special type of file which records how a solution is visualised. It is recommeded to always keep this file so that if the user ever needs a different format of visualisation, he can submit this VFG file directly without needing to solve the original problem again.

Disclaimer: The API should be considered to be in a very alpha phase at the moment. It's not expected to deal with heavy traffic and the error messages may not be complete. Until this disclaimer is removed, we do not recommend using the API for official business (academic or otherwise).

Thanks: The first version of Planimation API is developed by Changyuan Liu, Lingfeng Qiang, Mengyi Fan, Xinzhe Li and Zhaoqi Fang under guidance of Nir Lipovetzky. The python library is mainly written by Xinzhe Li and command line utility by Changyuan Liu.

API Tools

Python Library

The planimation_api.py library provides access to the API through the following functions (more to be added in the future):

Method Description
pddl_visualise(domain, problem, animation, format)Submit 3 PDDL files which corresponds to domain, problem, animation profile respectively and receive response in one of 5 supported formats (VFG, PNG, GIF, WEBM, MP4). Similar to "Build visualisation from problem" on user interface
vfg_visualise(vfg, format)Submit a VFG file and receive response in one of 4 supported formats (PNG, GIF, WEBM, MP4). Similar to "Build visualisation from solution VFG" on user interface

An example of usage is demonstrated below: 

import planimation_api as api

result = api.pddl_visualise("domain.pddl", "problem.pddl", "ap.pddl", "mp4")
print(result) // display the name of received file

Command-line Utility

The planimation.py command-line utility provides exactly same functionalities as the planimation_api.py library: 

> ./planimation.py

 planimation.py submitPDDL [path of domain] [path of problem] [path of animation profile] [output format]                 Submit 3 PDDL files which corresponds to domain, problem, animation profile respectively and receive response in one of 5 supported formats (VFG, PNG, GIF, WEBM, MP4)

 planimation.py submitVFG  [path of VFG] [output format]                Submit a VFG file and receive response in one of 4 supported formats (PNG, GIF, WEBM, MP4)
